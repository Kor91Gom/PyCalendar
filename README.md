    01 깃허브 코드스페이스 사용
    02 파이썬 버전인 Python 3.12.1
    03 streamlit 사용하여 웹페이지에서 실행하는 캘린더에 알맞은 알람을 만들고자 함.
    터미널로 실행 시 
        streamlit run calendar.py
        복사하여 Enter 키를 누르면 임시 웹을 실행할 수 있음


    🧠 기능적 특성
    기능                    설명
    월/연도 이동	            ◀ ▶ 버튼을 통해 이전/다음 달로 이동 가능하며, 1월/12월 경계에서는 연도도 자동 변경됨
    오늘 날짜 강조	        현재 날짜는 사용자 설정에 따라 배경색과 글자색으로 시각적으로 강조됨
    공휴일 표시	            한국 정부가 지정한 공휴일 및 대체공휴일은 빨간색 + 굵은 글씨로 표시되어 시각적으로 구분됨
    사용자 스타일 설정	    사이드바에서 배경색, 글자색, 글자 크기, 글자 두께를 직접 설정 가능
    반응형 UI 구성	        Streamlit의 columns를 활용해 요일 및 날짜를 그리드 형태로 배치함
    상태 유지	            st.session_state를 통해 사용자가 이동한 월/연도 상태를 유지하며, 새로고침에도 정보가 유지됨


    🎨 사용자 경험 요소
    직관적인 인터페이스: 월/연도 이동 버튼과 캘린더 헤더가 명확하게 배치되어 사용자가 쉽게 조작 가능
    시각적 커스터마이징: 색상과 글꼴을 자유롭게 설정할 수 있어 개인화된 캘린더 경험 제공
    공휴일 인식 강화: 중요한 날짜를 시각적으로 강조하여 일정 관리에 도움
    모바일/웹 호환성: Streamlit 기반이므로 다양한 디바이스에서 접근 가능


    🔧 확장 가능성
